# Server README

To connect to the MongoDB server, cd to server directory, and then run these </br>commands: </br></br>

Apply the Private Key: 
    
    export Brick_jwtPrivateKey=helloworld
   
Install Dependencies: 
  
    npm install

Connect to Database:   
    
    npm start
